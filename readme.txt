Install this plugin to add theBlonk sidebar. The theBlonk is a free blog promotion tool between participating blogs. 
== Description ==

theBlonk ("the blog link") is a tool that bloggers can use to help promote their blogs across the blogosphere. A blogger places a widget on their site that displays relevant posts from other bloggers, and in return their posts are displayed on other blogger's pages as well.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `theblonk.php` to the `/wp-content/plugins/theblonk/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the widget in a sidebar in the `widgets` menu in Wordpress
4. Enter your widget id from http://theBlonk.com


== Frequently Asked Questions ==

=How do I get a Widget Id=
Go to http://www.theBlonk.com to get a FREE widget id. A widget ID is required to complete the setup

= Is theBlonk Google-AdSense Safe? =

theBlonk's widget is compatible with google's adsense

= Whats the time investment? =

Less than 5 minutes! Sign up at theBlonk.com, install the plugin and enter your widget id. That's it!
